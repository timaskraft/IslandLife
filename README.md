IslandLife
----------
**Учебный проект второго модуля** [JavaRush - Java университет](https://javarush.com/university?utm_source=site&utm_medium=link_menu&utm_campaign=university)

ТЗ от ментора:

Необходимо запрограммировать модель острова с изменяемыми параметрами, состоящую из массива локаций (например, 100х20 клеток). Локации будут заполнены растительностью и животными. Животные могут:
есть растения и/или других животных (если в их локации есть подходящая еда),
передвигаться (в соседние локации),
размножаться (при наличии пары в их локации),
умирать от голода или быть съеденными.

- Архитектура через сервисы
- Lombok
- Jackson (сериализация/десериализация (YAML/JSON))
- Своя иерархия сущностей.
- Паттерн Фабрика
- Потоки
